# 🎬 S58_MOVIE_DIRECTORS

## 📌 Project Description
S58_MOVIE_DIRECTORS is a simple full-stack MERN application that allows users to manage a collection of movie directors. Users can add, update, and delete director details, which are stored in a database and displayed in a card format.

This project is my first MERN stack application, built to understand how frontend and backend systems interact.

---

## 🚀 Features
- ➕ Add new movie directors  
- 📝 Update existing director details  
- ❌ Delete directors  
- 📂 Store data in MongoDB  
- 🖼️ Display directors in card format  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- CSS  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB
